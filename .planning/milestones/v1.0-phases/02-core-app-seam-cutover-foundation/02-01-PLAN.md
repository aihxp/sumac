# Plan 02-01: Create the Core/App Runtime Seam

## Goal

Introduce `src/app/` as the new internal orchestration boundary for the golden
path and trim the `main.rs` match arms down to dispatch/runtime composition.

## Tasks

1. Add a new internal `app` module for golden-path orchestration.
2. Define runtime composition objects and request/result types for:
   - `setup`
   - `add`
   - `status`
   - `sync`
3. Move the current command-arm orchestration behind that seam.
4. Keep all current rendering and output behavior stable.

## Done When

- `src/app/golden_path.rs` exists
- `src/main.rs` command arms for the golden path delegate into the app layer
- no public CLI changes are introduced
