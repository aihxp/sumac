## v0.1.5 product note (not measured by this harness)

**MCP `stdio` / `http --list`:** servers that omit **prompts** or **resources** no longer fail the whole command when `prompts/list` or `resources/list` returns **`-32601` Method not found** — sxmc skips optional surfaces and exits **0** after tools (see [REAL_WORLD_SKILLS_AND_MCP_REPORT.md](REAL_WORLD_SKILLS_AND_MCP_REPORT.md)). This harness does not hit npm MCP servers; it only exercises nested **`serve`** and fixtures.

## Operational notes (benchmark maintenance)

| Issue | Resolution |
|-------|------------|
| **`git pull` / `origin/main` missing** | Upstream default branch is **`master`**; use `git fetch origin master:refs/remotes/origin/master` and track **`master`**. |
| **`sxmc scan` exits non-zero** when findings exist | Timing helpers must **not** treat that as a failed timing sample |
| **`git push` HTTP 403** with a restricted PAT | Use **`gh auth setup-git`** (or a token with **repo** scope) |
